
# Світ порід котів (CatTypes)

Інформаційний веб-сайт про різні породи котів: їх походження, характер, догляд та особливості. Проєкт демонструє адаптивний дизайн, інтерактивні елементи та зручну навігацію.

## Проєкт

Цей веб-сайт надає стислу, але корисну інформацію про різні породи котів, включаючи:

- Походження та історію порід.
- Особливості догляду.
- Популярність порід.
- Контактну форму для зворотного зв’язку.

Сайт має адаптивний дизайн і працює на десктопах та мобільних пристроях.

---

## Функціонал

- **Список порід:** картки порід із зображенням, описом та ключовими характеристиками.
- **Секція догляду:** список основних порад по догляду за котами.
- **Контактна форма:** можливість надсилати повідомлення (демо).
- **Анімації:** плавне масштабування картинок при наведенні та обертовий логотип; обертове лого біля назви сайту.

---

## Технології

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (анімація та обробка форми)
- Google Fonts (Inter)
- Адаптивний дизайн за допомогою медіа-запитів
